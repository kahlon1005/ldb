EBIZ istore - deployment steps
--------------------------------------

Copy java file from:
http://subversion.bcliquor.com/svn/eBiz/branch_projects/WMS/SourceCode/IStore/trunk/IStore/WebStore/src/oracle/apps/ibe/catalog/WOM_ibeProductAvailabilityWS.java


to the EBIZ istore server.

cp WOM_ibeProductAvailabilityWS.java $JAVA_TOP/classes/oracle/apps/ibe/catalog/

cd /opt/u01/womk1/fs2/EBSapps/comn/java/classes/oracle/apps/ibe/catalog/

javac WOM_ibeProductAvailabilityWS.java

run adadmin

to regenerate custom jar file

restart oacore managed servers for istore.
admanagedsrvctl.sh oacore_server3 and 4
cp $JAVA_TOP/oracle/apps/ibe/catalog/WOM_ibeProductAvailabilityWS.java $PATCH_BASE/EBSapps/comn/java/classes/oracle/apps/ibe/catalog/
cp $JAVA_TOP/oracle/apps/ibe/catalog/WOM_ibeProductAvailabilityWS.class $PATCH_BASE/EBSapps/comn/java/classes/oracle/apps/ibe/catalog/
